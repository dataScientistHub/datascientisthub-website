# DataScientistHub Website

Static website for DataScientistHub.

## Concept

DataScientistHub is a general platform for discovering, comparing, testing, and running trusted data science, automation, analytics, and AI capabilities from a curated library.

## Files

- `index.html` - Website content
- `styles.css` - Website styling
- `vercel.json` - Basic Vercel static hosting config

## Hosting

Upload these files to the `datascientisthub-website` GitHub repository, then import the repository into Vercel.
